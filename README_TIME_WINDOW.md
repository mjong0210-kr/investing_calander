# 조회 구간 기준

이 버전은 매일 06:00 KST 실행 시 아래 구간을 조회합니다.

```text
전일 08:00 KST ~ 실행 시각 KST
```

예: 2026-06-04 06:00 KST에 실행되면

```text
2026-06-03 08:00 KST ~ 2026-06-04 06:00 KST
```

`.env`에서 시작 시각을 바꿀 수 있습니다.

```env
RUN_TIME_KST=06:00
LOOKBACK_START_TIME_KST=08:00
```

주의: Investing.com의 `dateFrom/dateTo` 요청은 날짜 단위이므로, 코드는 먼저 전일~당일 데이터를 받은 뒤 Python에서 실제 시간 구간을 다시 필터링합니다.
